## 画像の推奨仕様

- **形式**: PNG（推奨）またはJPG
- **サイズ**: 推奨 400x600px 以上（アスペクト比 2:3）
- **ファイルサイズ**: 1枚あたり 100KB以下を推奨（Web表示のため）
- **背景**: 透明背景（PNG）または白背景（JPG）を推奨

## 画像の配置方法

1. 上記の命名規則に従って画像ファイルを準備
2. このディレクトリ（`public/images/tarot/`）に配置
3. アプリケーションで自動的に読み込まれます

## 注意事項

- ファイル名は正確に一致させる必要があります（大文字小文字も含む）
- 画像が存在しない場合、プレースホルダー（🃏）が表示されます
- アプリケーションは日本語のカード名から自動的に英語のファイル名に変換します

## 現在の実装

アプリケーションでは、日本語のカード名（例：「愚者」「ワンドのページ」）から自動的に英語のファイル名（例：`tarot-fool.png`、`tarot-wands-11.png`）に変換して画像を読み込みます。

変換ロジックは `app/Services/Fortune/TarotService.php` の `convertToEnglishFileName` メソッドで実装されています。
