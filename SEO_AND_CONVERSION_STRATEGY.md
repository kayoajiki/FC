# Fortune Compass SEO対策・離脱率最小化戦略

> **「検索で見つかりやすく、離脱せずにログインまで導く」設計**

---

## 🎯 現状の課題分析

### 現在の設計の問題点

1. **SEO対策が不十分**
   - ログイン前のコンテンツが薄い可能性
   - 静的ページ（記事・コラム）の構造が不明確
   - メタタグ・構造化データの実装方針が未定義

2. **離脱リスクが高いポイント**
   - ログイン前のエンゲージメントが低い
   - 診断結果を見た後、すぐに離脱する可能性
   - ログインへの心理的ハードル

3. **コンテンツマーケティングの不足**
   - 検索流入を獲得できる記事コンテンツが少ない
   - 内部リンク構造が弱い

4. **SNS・外部プラットフォームへの誘導が不足**
   - ブランディング・マーケティングの機会を逃している
   - 複数のチャネルでの世界観発信ができていない

## 🔍 SEO対策戦略

### 1. コンテンツ構造の最適化

#### 📄 静的ページの充実（ログイン不要）

```
【トップページ（/）】
├─ Heroセクション（「迷ったとき、自分に戻れる光を届ける」）
├─ 4占術の簡易説明（アンカーリンクで詳細ページへ）
├─ 生年月日入力フォーム（簡易診断）
└─ CTA：「もっと詳しく知る」→ 各詳細ページへ

【占い説明ページ（SEO重要）】
├─ /four-pillars（四柱推命とは）
├─ /ziwei（紫微斗数とは）
├─ /numerology（数秘術とは）
├─ /tarot（タロットとは）
└─ 各ページに：
   - 詳細な説明（2000-3000文字）
   - 祖乃果の世界観・哲学
   - 内部リンク（他の占術・コラムへ）
   - 診断フォームへの導線

【コラム・記事ページ（SEO最重要）】
├─ /column（コラム一覧）
├─ /column/{slug}（個別記事）
└─ 記事テーマ例：
   - 「30代で迷ったとき、占いが教えてくれること」
   - 「四柱推命で見る、あなたの強みの活かし方」
   - 「紫微斗数で知る、2025年の運勢の流れ」
   - 「数秘術で分かる、あなたの人生のテーマ」
   - 「タロットで読み解く、今日の小さな一歩」
   - 「祖乃果が語る、朝の光のような生き方」
   - 「自己理解を深める、3つの習慣」
   - 「迷いを整理する、占いの使い方」

【祖乃果プロフィールページ】
├─ /sonoka（祖乃果について）
└─ ストーリー・実績・世界観を深く伝える

【個別鑑定ページ】
├─ /consultation（個別鑑定について）
└─ STORESリンクへの導線

【物販ページ】
├─ /products（物販一覧）
└─ STORESリンクへの導線
```

#### 🎯 SEO対策の実装ポイント

1. **メタタグ・OGPタグの最適化**
   ```php
   // 各ページに適切なメタタグを設定
   - title: 「四柱推命とは | Fortune Compass - 迷ったとき、自分に戻れる光」
   - description: 「四柱推命で見る、あなたの本質と強み。Fortune Compassでは...」
   - og:image: 各ページに適切な画像
   - og:description: SNSシェア時の説明文
   ```

2. **構造化データ（JSON-LD）の追加**
   ```json
   {
     "@context": "https://schema.org",
     "@type": "WebSite",
     "name": "Fortune Compass",
     "description": "迷ったとき、自分に戻れる光を届ける占いプラットフォーム",
     "url": "https://fortunecompass.com",
     "potentialAction": {
       "@type": "SearchAction",
       "target": "https://fortunecompass.com/search?q={search_term_string}",
       "query-input": "required name=search_term_string"
     }
   }
   ```

3. **内部リンク構造の最適化**
   - 各ページから関連ページへのリンクを配置
   - パンくずリストの実装
   - サイトマップの自動生成

4. **コンテンツの質と量**
   - 各占術説明ページ：2000-3000文字
   - コラム記事：1500-2500文字
   - 定期的な更新（週1-2本の記事追加）

## 🚫 離脱率最小化戦略

### 1. ログイン前のエンゲージメント向上

#### 📍 段階的な価値提供（離脱を防ぐ仕組み）

```
【ステップ1】検索流入 → トップページ
┌─────────────────────────────────────┐
│ ✅ すぐに価値を感じられる            │
│   - Heroセクションで世界観を伝える   │
│   - 「今すぐ無料診断」CTA（目立つ）  │
│   - 生年月日入力フォーム（簡単）     │
└─────────────────────────────────────┘

【ステップ2】診断結果表示
┌─────────────────────────────────────┐
│ ✅ 結果を見た後も離脱させない        │
│   - 診断結果の一部を省略表示         │
│   - 「続きを見るにはログインが必要」 │
│   - 関連記事へのリンク（「あなたの強みを活かす方法」など）
│   - 他の占術も見てみる誘導          │
│   - コラム記事への誘導              │
└─────────────────────────────────────┘

【ステップ3】関連コンテンツ閲覧
┌─────────────────────────────────────┐
│ ✅ コンテンツを読んでからログイン誘導│
│   - 記事の途中・最後にログインCTA   │
│   - 「この記事を保存するにはログイン」│
│   - 「もっと詳しい診断結果を見る」   │
└─────────────────────────────────────┘
```

#### 🎯 離脱を防ぐ具体的な実装

1. **診断結果表示後のエンゲージメント**
   ```
   ✅ 診断結果カードの下部に：
      - 「あなたの強みを活かす方法」記事リンク
      - 「同じタイプの人の体験談」記事リンク
      - 「もっと詳しく知る」→ 占術説明ページへ
      - 「他の占術も見てみる」→ 他の占術診断へ
   
   ✅ スムーズなスクロール誘導：
      - 診断結果 → 関連記事 → ログインCTA
      - 途中で離脱しにくい流れ
   ```

2. **記事内のログイン誘導（自然な導線）**
   ```
   ✅ 記事の途中に：
      - 「この内容を保存するにはログイン」
      - 「もっと詳しい診断結果を見る」
      - 「今日の運勢をチェックする」
   
   ✅ 記事の最後に：
      - 「あなたの本質を知る、無料診断」
      - 「ログインして、毎日の運勢をチェック」
   ```

3. **エグジットインテント対策**
   ```
   ✅ ページを離れようとしたとき：
      - 「診断結果を保存しますか？」モーダル
      - 「メールアドレスを入力すると、結果を送信します」
      - ログインへの自然な導線
   ```

### 2. ログインへの心理的ハードルを下げる

#### 💡 ログイン前でも価値を提供

1. **診断結果の一部を表示**
   - 強み3つ + 注意1つは完全表示
   - 詳細な解説は「ログインして見る」に

2. **記事・コラムを無料で読める**
   - ログイン不要で記事を読める
   - 記事を読んでからログインしたくなる流れ

3. **「保存する」「詳しく見る」への自然な導線**
   - 「診断結果を保存する」→ ログイン
   - 「もっと詳しい診断結果を見る」→ ログイン
